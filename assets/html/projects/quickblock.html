QuickBlock

QuickBlock QuickBlock is a level-editing plugin for Unity, it is designed to create small-medium sized projects for developers and level designers. I created this project to help me learn more about Unity and C# programming. The project is still in development, but I am planning to release this for free. Note: This project is in it's early stages, so information might be outdated.

Features

  • Simple Building: Minecraft-Style building - use prefabs that snap together to make maps
  • Customisable: Use a range of colours and textures to specifically label each area of your map
