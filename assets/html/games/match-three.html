Match Three Fighter

Match Three Fighter is a match-three game where you battle enemy NPCs, and your goal is to survive. This project was a group project for university: I created the game's UI (but not assets), Menu Screen, Terrain, weather patterns, and mobile/phone movement. The reason I created this project was to improve my skills in porting games from different platforms.
